### Project Setup
#### 1\. Create a Virtual Environment
```bash
python -m venv .venv

.venv\Scripts\activate
```

-----

#### 2\. Configure API Key
```bash
copy .env.example .env
```

-----

#### 3\. Install Dependencies
```bash
pip install -r requirements.txt
```

-----

#### 4\. Run the Application
```bash
python main.py

streamlit run app.py
```
